# circle-rectangle-calculator
A simple C++ program that calculates the area and circumference of a circle and the area of a rectangle using a menu-based system.
Circle & Rectangle Calculator (C++)

This is a simple C++ program that calculates:

Area and Circumference of a Circle

Area of a Rectangle

The user selects an option from the menu and the program displays the result.

Features
Option 1 (Circle)

Radius is set to 5.6

Calculates:

Area

Circumference

Option 2 (Rectangle)

Length is set to 5.0

Width is set to 4.0

Calculates:

Area

How to Run
Compile
g++ main.cpp -o app

Run
./app
